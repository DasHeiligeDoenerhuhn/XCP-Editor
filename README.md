# What this thing can do:
Open a xcp Textfile in a GUI.
Just click onto Open and select a File.
# What this thing can't do:
Saving a edited Textfile
us either this:
  https://share.seanmcginty.space/?
or this:
  https://github.com/SnailMath/txt2xcp
for editing.
